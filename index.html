Distance Calculation Reference
Distance is measured on a staggered grid where adjacent keys horizontally or vertically are 1 unit apart, and the top (row 0) and bottom (row 2) rows are offset by 0.5 units horizontally relative to the home row (row 1). For each non-home key press, compute the straight-line (Euclidean) distance to the fingerâ€™s home key: 
distance = sqrt((row2 - row1)^2 + (col2 - col1)^2)
These distances are summed per finger to give the total travel effort. Diagonal moves are longer than purely horizontal or vertical moves since both row and column differences contribute.
